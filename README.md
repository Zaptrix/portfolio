# Don Cuddihee | Portfolio

A static portfolio for software, AI systems, technical support, networking, and visual design. It includes responsive light/dark themes and a theme-aware WebGL glass system core built with Three.js.

## Run locally

```powershell
npm start
```

Then open `http://localhost:8080`. You can also serve the repository directly with `python -m http.server 8080`.

## Verify

```powershell
npm run check
```

The committed WebGL bundle lets GitHub Pages serve the site directly from the repository root. After changing the scene source, rebuild it with `npm run build`.

## Structure

- `index.html`: content and semantic structure
- `src/css/style.css`: responsive visual system and motion
- `src/js/app.js`: theme control, navigation, and reveal effects
- `src/js/scene-source.js`: source for the live Three.js glass system core
- `src/js/scene.js`: browser-ready scene bundle generated by esbuild
- `Resume_Cuddihee.pdf`: current software and AI résumé

## Deployment

For GitHub Pages, set the publishing source to the repository's `main` branch and `/ (root)` directory. All asset paths are relative so the site also works from the existing `/portfolio/` project path.
